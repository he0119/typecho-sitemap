# Sitemap

Typecho 站点地图插件

地址：<https://domain.com/sitemap.xml>

## 安装

下载插件，修改文件夹为 Sitemap，上传到网站插件目录，后台启用。

```bash
cd typecho/usr/plugins
git clone https://github.com/he0119/typecho-sitemap.git Sitemap
```
